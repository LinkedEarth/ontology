Datasetc back to ToC or Class ToC

IRI: http://linked.earth/ontology#Dataset

A paleoclimate dataset containing measurements, interpretations, and metadata. Datasets form the core unit in the LinkedEarth ontology and include paleoclimate data (measurements), chronology data, physical sample information, and related publications.
Term rationale
Datasets are the core units of the LinkedEarth Ontology. Each Dataset corresponds to a paleoclimate archive (e.g., core, speleothem) and has descriptors (metadata) and data (measurements).
Example
http://wiki.linked.earth/MD982181.Khider.2014
is in domain of
has archive type op, has change log op, has chron data op, has collection name dp, has collection year dp, has compilation nest dp, has contributor op, has creator op, has data source dp, has dataset id dp, has funding op, has investigator op, has location op, has name dp, has notes dp, has original data URL dp, has paleo data op, has publication op, has spreadsheet link dp, has version dp

Interpretationc back to ToC or Class ToC

IRI: http://linked.earth/ontology#Interpretation

An interpretation of measured variables. This represents the climate or environmental interpretation derived from proxy measurements, including the relationship between the measured variable and the interpreted parameter.
Term rationale
An Interpretation connects proxies (e.g., Mg/Ca) to environmental/climate variables (e.g., temperature) and provides qualitative or quantitative information about this link.
Example
http://wiki.linked.earth/MD982181.Khider.2014.paleo1measurement2.sst.interpretation

is in range of
has interpretation op

Variablec back to ToC or Class ToC

IRI: http://linked.earth/ontology#Variable

A variable measured or derived in a dataset. Variables represent the actual measurements or values derived from the paleoclimate archive, and can include measured proxies, chronological controls, or interpreted climate parameters.
Term rationale
Variables are the basic units for analysis, including raw measurements (MeasuredVariable), modeled/interpolated values (ModeledVariable), and inferences of past climate (InferredVariable).
Example
http://wiki.linked.earth/MD982181.Khider.2014.paleo1measurement1.d18O

is in range of
has variable op
